Pertemuan 2: Data Spasial dan Non-Spasial

  • Definisi data spasial: Data yang berkaitan dengan lokasi dan bentuk di permukaan bumi.
  • Definisi data non-spasial: Data yang tidak memiliki informasi geografis.

Jenis Data Spasial

  • 1. Data Vektor: Titik, garis, dan poligon.
  • 2. Data Raster: Citra satelit, peta, dan grid.
  • 3. Metadata: Informasi tambahan tentang data spasial.

Pengumpulan Data Spasial

  • 1. Pengukuran lapangan.
  • 2. Citra satelit.
  • 3. Data sensus dan survei.

Pengolahan Data Non-Spasial

  • 1. Pengumpulan data dari sumber yang berbeda.
  • 2. Pemberian atribut pada data spasial.
  • 3. Penerapan data non-spasial pada analisis spasial.

Praktik: Mengimpor Data ke QGIS

  • Langkah-langkah untuk mengimpor data vektor dan raster ke dalam QGIS.
  • Menambahkan layer dan mengatur simbolisasi.

Studi Kasus: Pemetaan Wilayah Tertentu

  • Contoh penggunaan data spasial dalam pemetaan wilayah.
  • Analisis sebaran populasi, penggunaan lahan, dan sumber daya.
