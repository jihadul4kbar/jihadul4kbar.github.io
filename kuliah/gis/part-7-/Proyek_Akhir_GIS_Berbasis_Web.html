Pertemuan 7: Proyek Akhir GIS Berbasis Web

  • Proyek Akhir: Pembuatan aplikasi WebGIS sederhana.
  • Tujuan: Mengaplikasikan teori dan praktik GIS berbasis web.

Spesifikasi Proyek

  • Aplikasi harus menampilkan data spasial.
  • Fitur interaktif seperti marker dan layer harus ada.

Langkah-langkah Pembuatan Proyek

  • Langkah 1: Setup peta dasar menggunakan Leaflet atau OpenLayers.
  • Langkah 2: Menambahkan data spasial (contoh: file GeoJSON atau data marker).
  • Langkah 3: Menambahkan interaktivitas (zoom, popup, dll.).

Kriteria Penilaian

  • Tampilan dan interaktivitas peta.
  • Penggunaan data spasial dan tool GIS.
  • Dokumentasi proyek yang jelas.

Penutup

  • Persiapan untuk presentasi dan penilaian proyek.
