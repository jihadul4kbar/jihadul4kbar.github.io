Pertemuan 1: Pengenalan GIS

  • Materi: Pengertian dan sejarah GIS, Komponen-komponen GIS, Aplikasi GIS dalam berbagai bidang.
  • Rujukan: A Very Brief History of GIS, Buku: "Introduction to Geographic Information Systems" oleh Kang-Tsung Chang.
  • Contoh: Pengenalan aplikasi GIS di pemetaan dan analisis data.
  • Tugas: Buat esai pendek tentang manfaat GIS dalam kehidupan sehari-hari.

Pertemuan 2: Pengenalan Software GIS

  • Materi: Pengenalan QGIS dan ArcGIS, Instalasi dan antarmuka pengguna.
  • Rujukan: QGIS Training Manual, Esri Academy.
  • Contoh: Instalasi dan eksplorasi antarmuka QGIS.
  • Tugas: Instalasi QGIS dan ArcGIS, serta screenshot antarmuka pengguna.

Pertemuan 3: Data Spasial dan Sumbernya

  • Materi: Jenis-jenis data spasial (vektor dan raster), Sumber data spasial (satelit, survei lapangan, dll).
  • Rujukan: Buku: "Geographic Information Systems and Science" oleh Paul A. Longley et al., Getting Started with GIS.
  • Contoh: Contoh dataset spasial dari survei lapangan.
  • Tugas: Cari dan unduh dataset spasial sederhana.

Pertemuan 4: Pengelolaan Data Spasial

  • Materi: Import dan export data spasial, Format file data spasial (shapefile, GeoJSON, dll).
  • Rujukan: Buku: "GIS Fundamentals: A First Text on Geographic Information Systems" oleh Paul Bolstad, QGIS Documentation.
  • Contoh: Import dataset dari berbagai format file ke dalam QGIS.
  • Tugas: Import dataset yang diunduh ke dalam software GIS.

Pertemuan 5: Pengolahan Data Spasial

  • Materi: Edit dan manipulasi data spasial, Georeferensi data spasial.
  • Rujukan: Georeferencer Plugin, Introduction to GIS Data Manipulation.
  • Contoh: Edit dan georeferensi peta historis dalam QGIS.
  • Tugas: Lakukan edit dan georeferensi pada dataset yang diimport.

Pertemuan 6: Analisis Spasial Dasar

  • Materi: Teknik buffering dan overlay, Analisis spasial sederhana.
  • Rujukan: Buku: "Spatial Analysis and GIS" oleh David J. Maguire et al., Spatial Analysis with QGIS.
  • Contoh: Buat buffer dan overlay untuk analisis zonasi di QGIS.
  • Tugas: Buat buffer dan overlay pada dataset yang diimport.

Pertemuan 7: Visualisasi Data Spasial

  • Materi: Pembuatan peta tematik, Simbolisasi dan styling data spasial.
  • Rujukan: Making a Map in QGIS, Buku: "Designing Better Maps: A Guide for GIS Users" oleh Cynthia Brewer.
  • Contoh: Pembuatan peta populasi menggunakan QGIS.
  • Tugas: Buat peta tematik berdasarkan dataset yang diimport.

Pertemuan 8: Ujian Pertama

Pertemuan 9: Analisis Spasial Lanjutan

  • Materi: Teknik interpolasi dan analisis jaringan, Studi kasus analisis spasial lanjutan.
  • Rujukan: Buku: "Geospatial Analysis: A Comprehensive Guide" oleh Michael J. de Smith et al., Network Analysis with QGIS.
  • Contoh: Interpolasi data cuaca menggunakan QGIS.
  • Tugas: Lakukan interpolasi pada dataset yang diimport.

Pertemuan 10: Pemetaan dan Kartografi

  • Materi: Prinsip-prinsip kartografi, Layout peta dan elemen-elemen peta.
  • Rujukan: Buku: "Cartography: Thematic Map Design" oleh Borden D. Dent et al., Creating Maps in ArcGIS Pro.
  • Contoh: Buat layout peta topografi dengan QGIS.
  • Tugas: Buat layout peta lengkap dengan legenda, skala, dan judul.

Pertemuan 11: Integrasi Data Non-Spasial

  • Materi: Integrasi data tabel dengan data spasial, Join dan relate data.
  • Rujukan: Attribute Data in QGIS, Buku: "The GIS 20: Essential Skills" oleh Gina Clemmer.
  • Contoh: Join data demografi dengan data spasial di QGIS.
  • Tugas: Integrasikan data tabel dengan dataset yang diimport.

Pertemuan 12: Analisis Temporal dan Spatio-Temporal

  • Materi: Analisis perubahan temporal, Teknik analisis spatio-temporal.
  • Rujukan: Temporal GIS: Issues and Solutions, Buku: "Time in Geographic Information Systems" oleh Gail Langran.
  • Contoh: Analisis perubahan penggunaan lahan dari tahun ke tahun di QGIS.
  • Tugas: Lakukan analisis temporal pada dataset yang diimport.

Pertemuan 13: Penggunaan GPS dan Data Lapangan

  • Materi: Penggunaan GPS untuk pengumpulan data, Integrasi data GPS dengan GIS.
  • Rujukan: GPS and GIS Integration, Buku: "Fundamentals of GPS Receivers: A Hardware Approach" oleh James Bao-Yen Tsui.
  • Contoh: Menggunakan data GPS untuk pemetaan lokasi lapangan di QGIS.
  • Tugas: Ambil data GPS sederhana dan integrasikan dengan dataset yang diimport.

Pertemuan 14: Analisis 3D dan Pemetaan 3D

  • Materi: Pengenalan analisis 3D, Pembuatan model 3D.
  • Rujukan: 3D Analysis in QGIS, Buku: "3D GIS: A Geospatial Technology Handbook" oleh Bishop et al.
  • Contoh: Pembuatan model 3D dari data elevasi di QGIS.
  • Tugas: Buat model 3D dari data spasial yang diimport.

Pertemuan 15: Studi Kasus dan Aplikasi Nyata

  • Materi: Studi kasus aplikasi GIS di berbagai bidang, Diskusi kelompok tentang aplikasi GIS.
  • Rujukan: Applications of GIS, Buku: "GIS Applications in Agriculture" oleh Francis J. Pierce dan David Clay.
  • Contoh: Presentasi aplikasi GIS di bidang pertanian.
  • Tugas: Buat presentasi singkat tentang aplikasi GIS di bidang yang dipilih.

Pertemuan 16: Ujian Akhir

  • Materi: Pengenalan LeafletJS, Pengenalan Laravel, Integrasi Leaflet dengan Laravel.
  • Rujukan: Menampilkan Peta dengan LeafletJs, Membangun Web GIS Sekolah Dengan Laravel 8 Dan Leaflet.
  • Contoh: Integrasi dasar LeafletJS ke dalam aplikasi Laravel.
  • Tugas: Buat halaman web sederhana dengan peta interaktif menggunakan LeafletJS dan Laravel.

Pertemuan 18: Pengelolaan Data Spasial pada Web

  • Materi: Pengelolaan data spasial dalam Laravel, Integrasi data spasial dengan LeafletJS.
  • Rujukan: Cara Membuat Map di Laravel Melalui OpenStreetMap, GitHub - PAPATIRI/web-gis.
  • Contoh: Menampilkan data spasial dari basis data ke peta LeafletJS.
  • Tugas: Buat aplikasi web sederhana yang menampilkan data spasial dari basis data Laravel ke peta LeafletJS.

Pertemuan 19: Analisis Spasial pada Web

  • Materi: Analisis spasial sederhana pada web, Integrasi analisis spasial dengan Laravel.
  • Rujukan: Buku: "Spatial Analysis and GIS" oleh David J. Maguire et al., Spatial Analysis with QGIS.
  • Contoh: Analisis buffering dan overlay menggunakan LeafletJS.
  • Tugas: Buat aplikasi web yang menampilkan hasil analisis spasial pada peta LeafletJS.

Pertemuan 20: Pengembangan Aplikasi Web GIS

  • Materi: Pengembangan aplikasi web GIS lengkap, Studi kasus aplikasi GIS nyata.
  • Rujukan: Menampilkan Peta dengan LeafletJs, Buku: "GIS Applications in Agriculture" oleh Francis J. Pierce dan David Clay.
  • Contoh: Pengembangan aplikasi web GIS untuk bidang pertanian.
  • Tugas: Buat presentasi singkat tentang aplikasi GIS web yang dikembangkan.
